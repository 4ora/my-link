# [PRD] 마이링크 (MyLink) - 링크트리 클론 서비스

## 1. 프로젝트 개요
### 1.1 프로젝트명
- **마이링크 (MyLink)**

### 1.2 목적
- 사용자가 자신의 SNS 채널, 포트폴리오, 블로그 등의 다양한 링크를 하나의 페이지에서 효율적으로 보여줄 수 있는 개인 브랜딩 랜딩 페이지 제공.
- 링크트리(Linktree)의 핵심 기능을 벤치마킹하여 관리와 공유가 쉬운 서비스를 구축.

### 1.3 대상 사용자
- 인스타그램, 틱톡 등 바이오 링크가 하나만 허용되는 SNS를 사용하는 크리에이터.
- 자신의 포트폴리오를 간단한 페이지로 구축하고 싶은 구직자 및 학생.
- 다수의 온라인 채널을 운영하는 인플루언서 및 비즈니스 유저.

---

## 2. 핵심 기능 목록

### 2.1 필수 기능 (Essential - MVP)
- **프로필 관리**: 프로필 이미지 업로드, 닉네임 설정, 짧은 소개글 입력 기능.
- **링크 관리 (Link Builder)**:
    - 링크 추가: 제목 및 URL 입력.
    - 링크 수정 및 삭제.
    - 링크 순서 변경 (Drag & Drop 방식 지향).
- **고정 테마 시스템**: 전문가가 디자인한 5가지 이상의 고정 컬러 테마 패키지 제공.
- **모바일 최적화 랜딩 페이지**: 모바일 기기에서의 시인성과 터치 편의성을 최우선으로 한 반응형 페이지.

### 2.2 선택 기능 (Optional - 고도화)
- **Bento Box 레이아웃**: 현대적인 그리드 스타일 레이아웃 옵션 추가.
- **링크 클릭 통계**: 각 링크별 노출 수 및 클릭 수 추적 대시보드.
- **마이크로 인터랙션**: 링크 버튼 호버 및 클릭 시 부드러운 애니메이션 효과.
- **SNS 메타데이터 최적화 (SEO)**: 공유 시 카드 형태의 미리보기를 위한 OpenGraph 설정 지원.
- **다크 모드**: 테마별 다크 모드 토글 기능.

## 3. 기능 상세 설명

### 3.1 모바일 퍼스트 접근성
- 모든 UI 요소는 한 손 조작 범위(Thumb Zone)를 고려하여 배치.
- 버튼 크기는 최소 44x44px 이상으로 설정하여 터치 오류 방지.
- 텍스트 가독성을 위해 최소 16px 이상의 폰트 크기 사용.

### 3.2 고정 테마 시스템
- 사용자는 자유 색상 지정 대신, 미리 준비된 감각적인 테마 중 하나를 선택.
- 예시 테마:
    - **Neutral**: 깔끔한 화이트/그레이 조합.
    - **Midnight**: 깊은 네이비와 오렌지 포인트.
    - **Forest**: 차분한 다크 그린과 베이지.
    - **Candy**: 화사한 파스텔 핑크와 퍼플.
    - **Bento Dark**: 현대적인 블랙 그리드 스타일.

### 3.3 링크 유효성 검사
- `http://` 또는 `https://` 형식이 포함되어 있는지 실시간으로 확인.
- 올바르지 않은 형식 입력 시 에러 메시지 노출.

---

## 4. 제 제안 및 개선 사항
- **Bento Grid 활용**: 단순 리스트보다 더 풍부한 정보를 담을 수 있는 그리드 형태를 선택 기능으로 넣어 시인성을 높였습니다.
- **클립보드 복사 버튼**: 내 마이링크 주소를 원클릭으로 복사하여 SNS에 바로 붙여넣을 수 있는 편의 기능 추가를 권장합니다.
- **로딩 애니메이션**: 페이지 로드 시 스켈레톤 UI를 사용하여 사용자 이탈을 방지.
